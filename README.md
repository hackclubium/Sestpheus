# Sestpheus
A cool bot who loves sestinas!

Slack bot that watches opted-in channels for opted-in users. When a 39-line message follows the sestina end-word rotation, it reposts the text in-thread.

Commands: `/sest-in`, `/sest-out`, `/sest-chan-in`, `/sest-chan-out`. Response pattern: `User/channel opt-in/out to Sestpheus`.
